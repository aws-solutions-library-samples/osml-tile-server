aws.osml.tile_server.lambda.cleanup_dlq.lambda_handler(event: Dict[Any, Any], context: Any)[source]

This Lambda function processes SQS queue messages, retrieving and updating the associated items from a DynamoDB table. It expects an event object with a “Records” list which includes a ‘viewpoint_id’. If viewpoint_id is missing or item is not found in the table, it returns a 404 response. If the item is found, its status is updated as ‘FAILED’ with an expiry time of current time plus 1 day. If an exception occurs, it’s logged and a 500-status code is returned with the error message.

event = {

} context = {} result = lambda_handler(event, context)

print(result)

A dictionary containing the response status code and body.
