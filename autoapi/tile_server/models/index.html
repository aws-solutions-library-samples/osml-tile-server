class tile_server.models.ViewpointApiNames

Bases: str, aws.osml.tile_server.utils.string_enums.AutoLowerStringEnum

Provides enumeration of API names associated with a viewpoint.

These API names are used to map to specific operations on the viewpoint.

Variables:

  • UPDATE – Update viewpoint.

  • DESCRIBE – Describe viewpoint.

  • METADATA – Viewpoint metadata.

  • BOUNDS – Viewpoint bounds.

  • INFO – Viewpoint information.

  • TILE – Viewpoint tile.

  • PREVIEW – Viewpoint preview.

  • CROP – Crop viewpoint.

  • STATISTICS – Viewpoint statistics.

class tile_server.models.ViewpointStatus

Bases: str, aws.osml.tile_server.utils.string_enums.AutoUnderscoreStringEnum

Provides status options for a viewpoint that represents the state of processing or readiness for a viewpoint.

Variables:

  • NOT_FOUND – Viewpoint isn’t found.

  • REQUESTED – Viewpoint requested.

  • UPDATING – Viewpoint updating.

  • READY – Viewpoint ready.

  • DELETED – Viewpoint deleted.

  • FAILED – Viewpoint failed.
