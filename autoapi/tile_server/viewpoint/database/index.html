This is a helper class that extends json.JSONEncoder. It’s used to convert all Decimal instances to int when we fetch data from DynamoDB (which can return some items as Decimal).

Parameters:

json.JSONEncoder – Inherited JSON Encoder class from json module.

default(obj)

Overriden default method from JSONEncoder class. Converts all Decimal instances to int else returns default conversion.

Parameters:

obj – Object for JSON Encoding.

Returns:

Integer representation of Decimal else default JSON Conversion
