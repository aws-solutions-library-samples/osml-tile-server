Get a preview/thumbnail image in the requested format

Parameters:

  • aws – Injected AWS services.

  • viewpoint_id – Unique viewpoint id to get from the table.

  • img_format – The Desired format for preview output, valid options are defined by GDALImageFormats.

  • max_size – Max size of the preview image, defaults to 1024 pixels.

  • width – Preview width in pixels that supersedes scale if > 0.

  • height – Preview height in pixels that supersedes scale if > 0.

  • compression – GDAL image compression format to use.

Returns:

Response of preview binary with the appropriate mime type based on the img_format
